# TRAVEL DIARY

## ENV Variables
This project uses the following invironment variables

```
BASE_URL=...
PORT=...

// MYSQL
MYSQL_HOST=...
MYSQL_USER=...
MYSQL_PASSWORD=...
MYSQL_DATABASE=...

// bcript
SALT_ROUNDS=...

// SMTP
SMTP_HOST=...
SMTP_PORT=...
SMTP_USER=...
SMTP_PASSWORD=...
SENDER_EMAIL=...

// JWT
JWT_PRIVATE_KEY=...
JWT_EXPIRES_AFTER=...
```

## Postman
In `/postman` folder you can find the **postman** json collection. Install it in your **postman** to test the endpoints for this project
